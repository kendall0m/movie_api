URL endpoint Request Method Response
/movies GET return a JSON object of all movies
/movies/:title GET return a JSON object holding data about a single movie by title
/movies/genre/:genreName GET return a JSON object holding data about a genre by genre name
/movies/directors/:directorName GET return a JSON object holding data of a director by name
/users POST allow new users to register; responds with a test message indicating successfully signing up
/users/:id/ PUT allow users to update their username; return a message indicating the username was successfully changed
/users/:id/:movieTitle POST allow users to add a movie to their favorites list; returns a message whether the addition was successful
/users/:id/:movieTitle DELETE allow users to remove a movie from their favorites list; returns a message whether the removal was successful
/users/:id DELETE allow existing users to deregister; return a message confirming the account was deleted
